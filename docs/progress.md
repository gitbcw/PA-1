#### 1. 基础架构

  - 路由保护
- 初始化项目结构
- 创建基本文档

  - 响应式设计，支持移动端
  - 分类展示菜单项
  - 支持折叠/展开

#### 2. 数据库集成
- 使用 Prisma ORM 连接 PostgreSQL
- 实现数据模型
  - 用户模型
  - 任务模型
  - 目标模型
  - 消息模型
  - 分类和标签模型
- 实现 API 路由
  - 用户 CRUD 操作
  - 任务 CRUD 操作
  - 目标 CRUD 操作

#### 3. LLM集成
- 支持OpenAI API
- 支持Anthropic API
- 支持智谱AI GLM-4 API
  - 专门为中文优化的大语言模型
  - 支持流式输出
  - 支持多轮对话
- 支持自定义LLM API
  - 可连接到任何兼容的LLM服务
  - 支持流式输出
  - 支持自定义请求头和参数
